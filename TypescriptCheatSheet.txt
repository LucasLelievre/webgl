== INSTALL TYPESCRIPT ==

1 - install npm on the web

2 - install typescript with npm in powershell

npm install -g typescript

3 - check typescript version

tsc --version

4 - Intialise typescript project

tsc --init

== COMPILE CODE ==

1 - just call tsc in the project directory

tsc

2 - cofiguration can be found in the tsconfig.json file
